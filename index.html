<h1>LED Image Viewer</h1>

Click me to upload an image!
  • A fun winter project which I've completed a prototype for, and am working on expanding using more panels to use for workshops with the Mobi organization at UTA.
  • This Raspberry Pi project uses an LED Matrix to display images uploaded from my website. I allow anyone to upload any image file (.png, .jpg, .jpeg, .gif) they would like and I manually accept appropriate images to be downloaded.
  • The Pi uses Secure Copy to download images from my server. After checking to see if they haven't already been downloaded, I use the hzeller rpi-rgb-led matrix repository to display and format images onto the matrix.
  • The working prototype is a 32x64 resolution spread across 6x12 inches. Mobi and I are looking to expanding it to a 128x128 resolution to display weather data, school news, and logos from our office window and for other educational/CS related uses.
